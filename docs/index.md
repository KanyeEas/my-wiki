# Welcome to My Wiki

This is the homepage of my new personal wiki, built with **MkDocs** and **Material for MkDocs**.

## Getting Started

You can edit this file at `docs/index.md` to change the content of the home page.

## Commands

* `mkdocs serve` - Run the development server
* `mkdocs build` - Build the documentation site
* `mkdocs help` - Print this help message

## Project Layout

    mkdocs.yml    # The configuration file.
    docs/
        index.md  # The documentation homepage.
        ...       # Other markdown pages, images and other files.
